## 📝 3 Steps to Start:

### 1️⃣ Create `.env` file
Create a file called `.env` in your project root:
```
VITE_GROQ_API_KEY=paste_your_key_here
```

### 2️⃣ Get FREE Groq API Key
- Go to: **https://console.groq.com/keys**
- Sign up (free!)
- Click "Create API Key"
- Copy it

### 3️⃣ Paste & Run
- Paste the key into `.env` file
- Run: `npm run dev`
- Done! 🎉
